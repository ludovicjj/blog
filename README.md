#Comment installer le projet :

-1 Copier tous les fichers et dossiers
-2 Suivre les instruction de configuration pour la base de donnée 

## Configuration de la base de donné

-1 Créer une base de donnée avec le nom que vous souhaiter
-2 Importer dans cette base le fichier blog.sql
-3 Dans le dossier config ouvrez le fichier config.php
-4 Vous devriez obtenir un tableau avec les champs suivant :
	
'db_user' => 'root',
'db_pass' => '',
'db_host' => 'localhost',
'db_name' => 'blog'

-5 Modifier les valeurs par les configuration de votre systeme de gestion de base de donnée
